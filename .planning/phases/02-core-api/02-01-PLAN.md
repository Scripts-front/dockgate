**GET /:name/latest handler:**

```typescript
appsRouter.get('/:name/latest', validateAppName, async (req, res) => {
  const { name } = req.params
  try {
    const stream = await minioClient.getObject(config.minioBucket, latestKey(name))
    const chunks: Buffer[] = []
    await new Promise<void>((resolve, reject) => {
      stream.on('data', (chunk: Buffer) => chunks.push(chunk))
      stream.on('end', resolve)
      stream.on('error', reject)
    })
    const manifest: LatestManifest = JSON.parse(Buffer.concat(chunks).toString('utf8'))
    res.json(manifest)  // D-03: return content directly, no envelope
  } catch (err) {
    if (isObjectNotFound(err)) {
      res.status(404).json({ error: 'No versions published' })  // D-04
      return
    }
    throw err  // let Express 5 global error handler return 500
  }
})
```

No auth middleware on this route — READ-01 specifies public access (no DOWNLOAD_TOKEN required for /latest).

**Handler implementation:**

```typescript
appsRouter.get('/:name/download', validateAppName, requireDownloadToken, validateVersion, async (req, res) => {
  const { name } = req.params
  const version = req.query.version as string

  // READ-03: verify .tar exists before generating presigned URL
  try {
    await minioClient.statObject(config.minioBucket, tarKey(name, version))
  } catch (err) {
    if (isObjectNotFound(err)) {
      res.status(404).json({ error: `Version ${version} not found` })  // D-09: literal version in message
      return
    }
    throw err  // non-404 S3 error or network failure — let global handler return 500
  }

  const rawUrl = await minioClient.presignedGetObject(config.minioBucket, tarKey(name, version), 3600)  // D-02: 3600s download
  const url = rewritePresignedUrl(rawUrl)  // replace internal hostname with MINIO_PUBLIC_ENDPOINT
  res.json({ url })  // D-01: only { url: string }, no expiresIn or other fields
})
```

**Note on statObject + presignedGetObject:** Two separate SDK calls. statObject confirms the object exists (throws on miss). presignedGetObject then generates the URL. This is intentional — presignedGetObject does NOT verify existence; it signs a URL for whatever key is provided, even if the object is absent.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Internet → GET /latest | Unauthenticated public endpoint; any caller may read latest.json content |
| Internet → GET /download | Client presents Bearer token or ?token= query param; token validated with timingSafeEqual |
| API → MinIO (internal Docker network) | API is trusted actor; MinIO network is internal only |
| MinIO presigned URL → Client | Client uses URL directly; MinIO validates AWS SigV4 signature on the URL |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Spoofing | GET /download — token validation | mitigate | requireDownloadToken uses timingSafeEqual(sha256(input), sha256(expected)) — constant-time comparison prevents timing oracle (SEC-01) |
| T-02-02 | Tampering | MinIO object key construction | mitigate | validateAppName + validateVersion reject any input outside `[a-zA-Z0-9._-]+` and strings containing `..` — prevents path traversal in MinIO keys (SEC-03) |
| T-02-03 | Information Disclosure | Presigned URL expiry | accept | URLs expire in 3600s; clients are expected to use immediately; short window limits exposure if URL is leaked |
| T-02-04 | Information Disclosure | GET /latest response | accept | latest.json contains only version metadata (no secrets); public endpoint is intentional design choice |
| T-02-05 | Elevation of Privilege | GET /latest — no auth check | accept | latest.json contains only public version metadata; no sensitive data; REQUIREMENTS.md explicitly states "no auth required" for READ-01 |
| T-02-06 | Denial of Service | GET /latest — stream collection from MinIO | accept | latest.json is a small JSON manifest (~200 bytes); no size limit needed; internal API with no public internet exposure |
| T-02-07 | Information Disclosure | ?token= in query string | mitigate | requestLogger middleware (from Phase 1, src/middleware/log.ts) already redacts query params in all request logs (SEC-05) |
</threat_model>
